This document provides a comprehensive reference of all keyboard shortcuts available in Traktor Bridge 2.0, helping you navigate the application efficiently and boost your productivity.

## Main Window

| Shortcut | Action |
|----------|--------|
| Ctrl+O | Open NML File |
| Ctrl+R | Reload Collection |
| Ctrl+Return | Start Conversion |
| Ctrl+L | View Log |
| Ctrl+D | View Playlist Details |
| F1 | Show Documentation |
| Alt+F4 | Exit Application |

## Details Window

| Shortcut | Action |
|----------|--------|
| P | Play/Pause Selected Track |
| Esc | Close Window |
| Ctrl+F | Focus Search Field |

## Timeline Dialog

| Shortcut | Action |
|----------|--------|
| C | Copy Cue Points to Clipboard |
| Esc | Close Dialog |
